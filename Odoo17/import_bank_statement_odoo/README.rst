Import Bank Statement
=====================
This module import bank statement in CSV, XLSX, OFX and QIF format.

Configuration
=============
- www.odoo.com/documentation/16.0/setup/install.html
- Install our custom addon
- Install openpyxl, ofxparse, qifparse
